## Laravel + Swoole in docker
you need to download laravel project in src folder and install swoole package

## YAML
- NGINX (latest)
- PHP-FPM (7.4.6)
- Laravel (8.x)

## Usage
```
make setup
make install
make run

#to stop
make stop

#to restart
make restart

#to destroy
make destroy
```
