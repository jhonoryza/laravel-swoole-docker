## Laravel + Swoole in Docker
already install package [laravel-swoole](https://github.com/swooletw/laravel-swoole)

## Requirement
- docker & docker-compose installed
- make installed

## Stack
- NGINX (latest)
- PHP-FPM (7.4.6)
- Laravel (8.x)

## Usage
```
#to initialized
make setup
make install

#to start
make run

#to stop
make stop

#to restart
make restart

#to destroy
make destroy
```
